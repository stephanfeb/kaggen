# System Information

Use this skill to get a quick summary of the system's current status.

## Available Commands

### get_summary.sh — Get System Summary

Returns a formatted report including the current date, system uptime, memory usage, and disk usage for the root partition.

```bash
bash scripts/get_summary.sh
```
